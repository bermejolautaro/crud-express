# Crud-Express

Crud app with expressjs as a backend and reactjs as frontend with jest for testing

## Install dependencies

```
npm ci
```

## Run application

```bash
npm run dev
```

## Seed database
```
npm run seed-db
```

## Testing
```
npm test
```
