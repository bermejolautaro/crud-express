# Crud-Express

Crud app with expressjs as a backend and reactjs as frontend with jest for testing

## Install dependencies

```
npm ci
```

## Run application

```bash
npm run dev
```

- The api is behind the route "/api"  
- The frontend is behind the route "/react"

## Seed database
```
npm run seed-db
```

## Testing
```
npm test
```
